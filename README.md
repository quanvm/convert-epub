Convert EPUB
====

## Install
Install [Calibre](https://calibre-ebook.com/download_osx) for MacOS

```bash
brew install php@7.4 yq
composer install
```

## Download
- Define a list in config.yml
- Execute download.sh
```shell
./download.sh
```
